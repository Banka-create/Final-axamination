# Task Manager

**Автор:** Адиль Сиражеев

## Описание программы

Консольное приложение для управления ваших задач с функциями:
- Создания обычных и срочных задач
- Установки приоритетов (низкий, средний, высокий)
- Отслеживания статуса (ожидает, в процессе, выполнена)
- Поиска и фильтрации задач
- Автоматического сохранения/загрузки в JSON файл
- Валидации пользовательского ввода
- Объектно-ориентированной архитектуры

## Требования

- Python 3.8 или выше
- Операционная система: Windows, macOS, Linux
